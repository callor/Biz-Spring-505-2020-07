## 첨부파일이 있는 게시판
* 2개의 첨부파일까지 있는 게시판 작성

### INSERT를 수행할때
* 첨부된 파일을 파일이름을 변경하여 upload를 수행하고
* 변경된 파일이름을 VO에 저장한 후 Table에 insert

### UPDATE를 수행할때
* 만약 새로운 파일이 업로드 되면  
	기존 파일을 삭제하고, 업로드를 수행
* 만약 새로운 파일이 선택되지 않으면  
	기존 파일은 그대로 유지한 상태로 Update만 수행
	
### DELETE를 수행할때
* 먼저 첨부파일을 모두 삭제한 후
* Table의 데이터를 삭제
